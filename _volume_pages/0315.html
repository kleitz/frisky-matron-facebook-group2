{% raw %}CHAPTER XXV{% endraw %}
{% raw %}AN unexpected DISCOVERY.{% endraw %}
{% raw %}Porthos and Aramis walked up Regent Street. It{% endraw %}
{% raw %}was their first visit to London. They were pleased{% endraw %}
{% raw %}and yet disappointed ; they saw much to admire, but{% endraw %}
{% raw %}the bright memory of their own city was not{% endraw %}
{% raw %}dimmed by the contrast. Even as they strolled{% endraw %}
{% raw %}slowly along, a November fog, a 'London particular,'{% endraw %}
{% raw %}foflowed them, engulfed them, and swept stealthily{% endraw %}
{% raw %}on towards the higher ground.{% endraw %}
{% raw %}' This is unfortunate,' said Aramis ; ' for, much as{% endraw %}
{% raw %}we hear of these sooty mists, yet Athos informs me{% endraw %}
{% raw %}that a real specimen fog only occurs half a dozen{% endraw %}
{% raw %}times a year.'{% endraw %}
{% raw %}' But do you not rather think we are very{% endraw %}
{% raw %}fortunate ?' said Porthos. ' Here we are, already{% endraw %}
{% raw %}two days in this city. We leave in a few hours.{% endraw %}
{% raw %}Fancy having to bid good-bye to London without{% endraw %}
{% raw %}having seen its particular speciality, the one gift in{% endraw %}
{% raw %}which it is supreme, the one production in which it{% endraw %}
{% raw %}excels all cities, ancient and modern.'{% endraw %}
{% raw %}' Well, I could wiflingly have postponed this{% endraw %}
{% raw %}pleasure until my next visit. Porthos, I can see{% endraw %}
{% raw %}you, but not well. Can you see me ? I am not so{% endraw %}
{% raw %}large as yourself, nor do I possess an outline so{% endraw %}
{% raw %}distinct.'{% endraw %}
