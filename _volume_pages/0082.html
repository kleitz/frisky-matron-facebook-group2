{% raw %}8o A FRISKY MATRON{% endraw %}
{% raw %}On the morning of her departure Mrs. Tacker sat{% endraw %}
{% raw %}in her drawing-room. A shrewd observer could{% endraw %}
{% raw %}have guessed that she was expecting someone to{% endraw %}
{% raw %}call by appointment. The door opened, and Mr.{% endraw %}
{% raw %}Dunne was announced,{% endraw %}
{% raw %}* Ah,' said she, extending her hand, ' it is so good{% endraw %}
{% raw %}of you to come, and I am so glad to meet you.{% endraw %}
{% raw %}Whom do I see to-day, Valentine or Orson ?'{% endraw %}
{% raw %}' To be Orson is my destiny,' said Dunne.{% endraw %}
{% raw %}* To all except to me. But why do you assume{% endraw %}
{% raw %}these brusque manners — you who can be so{% endraw %}
{% raw %}different ?'{% endraw %}
{% raw %}' Well, my fate was for many years cast among{% endraw %}
{% raw %}the hardest and the coarsest of men—among miners,{% endraw %}
{% raw %}cowboys, backwoodsmen, the roughest of the rough,{% endraw %}
{% raw %}but very honest and very true, most of them. Now,{% endraw %}
{% raw %}my theory is, that if a round man finds himself in a{% endraw %}
{% raw %}square hole, he ought to develop angles.'{% endraw %}
{% raw %}' Ah,' said Mrs. Tacker. ' It seems as if you had{% endraw %}
{% raw %}completely forgotten me and those past days, so few,{% endraw %}
{% raw %}and so sweet.'{% endraw %}
{% raw %}' You I shall never forget; but days like those are{% endraw %}
{% raw %}not meant to endure for ever.'{% endraw %}
{% raw %}' You have consoled yourself,' said she.{% endraw %}
{% raw %}' And you also.'{% endraw %}
{% raw %}' Yes ; but why should you and I not still be friends ?{% endraw %}
{% raw %}You know what the song says :{% endraw %}
{% raw %}"You should never be off with the old love{% endraw %}
{% raw %}Until you're well on with the new."{% endraw %}
