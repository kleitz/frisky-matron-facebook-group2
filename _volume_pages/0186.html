{% raw %}184 A FRISKY MATRON{% endraw %}
{% raw %}' I'll go bail she's got lots of money,'{% endraw %}
{% raw %}A train ran in, with a long string of carriages{% endraw %}
{% raw %}fiUed with Russian Jews. They looked spectral{% endraw %}
{% raw %}and as if in the last stages of famine, with wild eyes{% endraw %}
{% raw %}ravenously seeking for food, skeleton faces, and rags{% endraw %}
{% raw %}and filth unutterable. They looked like a hoard of{% endraw %}
{% raw %}famished savage fleas leaping to fasten on John Bufl.{% endraw %}
{% raw %}' I must not stay much longer,' said Mrs. Tacker{% endraw %}
{% raw %}as the train moved off. ' I want to be early at the{% endraw %}
{% raw %}dance.'{% endraw %}
{% raw %}' What dance ?' said Sir Louis.{% endraw %}
{% raw %}' The dance at Woolwich.'{% endraw %}
{% raw %}' At Woolwich ? Then we had better look sharp.{% endraw %}
{% raw %}We are not far from Woolwich here, as the crow{% endraw %}
{% raw %}flies; but to get to it we have to travel up this side{% endraw %}
{% raw %}by the Thames for miles and miles, cross over to{% endraw %}
{% raw %}another line, wait perhaps an hour for a train, and{% endraw %}
{% raw %}then down the other side of the river, miles and{% endraw %}
{% raw %}miles again, to Woolwich. There are no bridges down{% endraw %}
{% raw %}here, remember.'{% endraw %}
{% raw %}They hurried into the hotel, and found there was{% endraw %}
{% raw %}no train up from Purfleet for six hours. Very few{% endraw %}
{% raw %}trains stop there.{% endraw %}
{% raw %}Mrs. Tacker was in despair.{% endraw %}
{% raw %}' Oh, Louis !' said she, ' can't you do something ?{% endraw %}
{% raw %}I have never been at a gunner's dance in England,{% endraw %}
{% raw %}and theirs is always the best of all; and the Knight-{% endraw %}
{% raw %}Bartons will be there, and that odious Mrs. Bellairs,{% endraw %}
{% raw %}and everybody. And, besides, I want to go most{% endraw %}
