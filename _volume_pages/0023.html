{% raw %}CHAPTER II.{% endraw %}
{% raw %}BOHEMIA.{% endraw %}
{% raw %}The end of dinner found Yussuff in a most con¬{% endraw %}
{% raw %}tented state of mind. He found stealing over him{% endraw %}
{% raw %}a spirit of vivacity, a feeling of elan, which would{% endraw %}
{% raw %}carry him through anything. The opera was ' La{% endraw %}
{% raw %}Muette de Portici,' or ' Masaniello.' The plot in¬{% endraw %}
{% raw %}cludes a revolution and many murders, and among{% endraw %}
{% raw %}the two thousand auditors around there must have{% endraw %}
{% raw %}been many hundreds of revolutionists, and probably{% endraw %}
{% raw %}a few murderers, Paris has its realisms. The{% endraw %}
{% raw %}ballet was superb. There is only one ballet worth{% endraw %}
{% raw %}the time spent in looking at it, and that is at the{% endraw %}
{% raw %}Grand Opera. Afterwards a rapid drive brought{% endraw %}
{% raw %}the friends to the ball of all the talents, and Yussuff{% endraw %}
{% raw %}looked on, while the musketeers were quickly among{% endraw %}
{% raw %}the dancers.{% endraw %}
{% raw %}Quadrilles were in progress, and wonderful were{% endraw %}
{% raw %}the steps introduced. A great many students were{% endraw %}
{% raw %}present, and they showed off their activity in those{% endraw %}
{% raw %}marvellous gymnastics which a stranger may marvel{% endraw %}
{% raw %}at, but cannot hope to rival. Some remarkable{% endraw %}
{% raw %}feats of agility were performed. Yussuff looked on{% endraw %}
{% raw %}deeply interested until the music ceased.{% endraw %}
{% raw %}' Hola, mon ami, you are not dancing 1' said{% endraw %}
{% raw %}Aramis, coming up with a very pretty girl on his{% endraw %}
{% raw %}arm. ' Come, I will get you a partner. By the{% endraw %}
{% raw %}way, does monsieur waltz ?'{% endraw %}
